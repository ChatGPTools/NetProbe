# NetProbe

NetProbe è uno strumento Python con interfaccia grafica che combina funzionalità di web scraping e analisi di rete. Consente agli utenti di estrarre facilmente titoli da URL forniti e ottenere informazioni dettagliate sugli indirizzi IP, inclusi dettagli su IP pubblici, interni e la verifica se un indirizzo è un bogon.

## Funzionalità

- **Estrazione Titoli**: Inserisci un URL per estrarre i titoli delle notizie presenti sulla pagina.
- **Analisi IP**: Ottieni informazioni dettagliate su un indirizzo IP specifico, compreso se si tratta di un indirizzo bogon.
- **Rilevamento IP**: Visualizza l'indirizzo IP pubblico del server che ospita un sito web e l'indirizzo IP interno del computer utilizzato.

## Requisiti

- Python 3.x
- Requests: `pip install requests`

## Uso

1. Clona il repository o scarica i file sorgente.
2. Installa le dipendenze necessarie.
3. Esegui `NetProbe.py` per avviare l'applicazione.
